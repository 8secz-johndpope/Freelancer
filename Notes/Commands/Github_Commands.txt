Add new project to Github: 
--------------------------
1. Manually create remote repository using "Add Repository" option on GUI. It will also show steps to add it.
 
2. Now go to project directory on local machine and follow steps as below:
git init
git add .  If need to undo this then (git reset . or git reset filename)
git commit -m "first commit"
git remote add origin https://github.com/poojagpta/HadoopPurchaseTrend.git
git push -u origin master 
Will ask for username/password (poojagpta/....)

Add new project with remote repository created to Github using command Line:
----------------------------------------------------------------------------
curl -u 'poojagpta' https://api.github.com/user/repos -d "{\"name\":\"HbaseDemo\"}"

git remote add origin https://github.com/poojagpta/HbaseDemo.git


Create branch:
--------------
git checkout -b branch1 
git branch -a  
git push origin branch1

Remove file:
-----------
git rm -r src/

Undo your local changes:
-------------------------
git stash (Save for later usage)
git checkout -- <file name>
git reset . (remove all add in files and we need to add in files again)
git reset --hard (discard all local changes)

Merge the files from branch to master:
----------------------------------------
For fast-forward merge, use below command:
git checkout master
git merge --no-ff test

Look into changes in fast-forward merge and then commit the code

If we want to take all changed from test branch and then merge to master use below command:
git merge --strategy-option theirs test -m "merge test to master"

Tagging a branch:
==================
git tag -a v1.4 -m "my version 1.4"
git push origin v1.4

Delete a tag:
git tag -d v1.4
git push origin :refs/tags/v1.4
